<< Usage >>
1. Create a material with 'MeltShader' or 'MeltShader (Normal Map)' shader.
2. Set that material to meshes.
3. Change parameters.
4. Enjoy!

<< Parameters >>
* Tilt angle ... Max tilt angle when melting.
* Spread power ... Spread XZ ratio in fading.
* Melt color ... Color of melted.
* Melt rate ... Fading rate. You control this parameter in scripts.
* Object height ... Set the fade object height.
* Object melt height ... Set the melt height.
* Object base position ... Set the object position. If the object moving, you have to reset this parameter.
